# vkRender

根据[Vulkan Tutorial](https://docs.vulkan.net.cn/tutorial/latest/00_Introduction.html) 学习并构建这个渲染引擎。

一个基于Vulkan + Glfw + glm 构建的一个渲染引擎
可能内置一个自己写的DeviceMemory分配器...
